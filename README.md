# Square Dancing Choreography MCP Server

Encodes square dancing choreography structures using category theory for visualization.

## Installation

```bash
pip install -e ".[dev]"
```

## Testing

```bash
./tests/run_tests.sh
```

## Structure

- 9 dimensions with 50+ tokens
- 4 categorical structures (monoid, groupoid, category, functor)
- Visualization prompt generation for choreography diagrams

See full documentation for details.
